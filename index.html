About

Guitab Writer is a Graphical User Interface Tab Writer that allows users to view a virtual Fretboard, allowing them to click on frets and strings so that they can build their tab for songs.

Guitab Writer was made for my Third Year Project for the Module Professional IT Skills. I used a template from a Bootstrap template.
